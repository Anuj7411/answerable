# @answerfox/core

Shared types, utilities, and error classes for the [Answerfox](https://github.com/Anuj7411/answerfox) SEO toolkit.

> **v0.1.0.** Branded URL types, the `Check<T>` interface, and the error hierarchy ship today. Stable API surface — only additive changes before v1.0.

## Install

```bash
pnpm add @answerfox/core
```
